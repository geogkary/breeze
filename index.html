Build JSON APIs

Use Breeze to quickly setup and start building JSON APIs with PHP. Breeze is a lightweight framework that takes care of versioning and routing, built on top of mikecao/flight.

Quickly setup and start building

Versioning

Handle your APIs through a simple versioning system, designed to help you manage and build more efficiently.

Routing

Breeze is built on top of mikecao/flight, which means everything is routed automatically.

Lightweight

There's absolutely no ninja magic involved and no obfuscated processes.

Simple

No hard stuff to learn and no re-inventing the wheel, except the things you don't need to know about.
