Get the boring stuff out of the way

Configuration

Your web applications and websites are basically themes, pre-packed with configuration options.

Routing

Webmart is built on top of mikecao/flight, which means everything is auto-routed automatically.

Toolkit

Store and retrieve data, load CSS/JS assets, Bootstrap, create forms and more, with a simple toolkit.

MVC

Create full-fledged web applications and websites, with a fully supported MVC logic.

Libraries

Webmart supports a few libraries, but you can also load and use your own.

Install

For a manual setup, see the README on GitHub.

If you're using Composer, require webmart/webmart, create an index.php file and include vendor/autoload.php.

Open in your browser.

Contribute

Webmart was created out of frustration - building static websites is painstakingly too simple and loading a CMS is ridiculously too complex.

So basically, copying and pasting the same template for different projects turned into an automated solution.

It's of course an open source project, and everyone is more than welcome to contribute.

Feel free to fork the repo on GitHub.
